Summary

The header <stdexcept> is part of the Diagnostics library of the Standard C++ Library. It defines the classes that the Standard C++ Library itself and C++ programs may use to report certain errors. These classes include: logic_error, domain_error, invalid_argument, length_error, out_of_range, runtime_error, range_error, overflow_error, and underflow_error.
