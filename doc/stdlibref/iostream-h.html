Summary

The header <iostream> is part of the Input/output library of the Standard C++ Library. It declares the eight standard iostream objects: cin, cout, cerr, clog, wcin, wcout, wcerr, and wclog.

Synopsis

 
namespace std {
  extern istream cin;
  extern ostream cout;
  extern ostream cerr;
  extern ostream clog;

  extern wistream wcin;
  extern wostream wcout;
  extern wostream wcerr;
  extern wostream wclog;
}
